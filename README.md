# metronome
Web-based metronome with beat counter and song charting.


## Getting Started
```bash
git clone git@github.com:sonph/metronome.git
cd metronome
yarn install
```

## Testing
- Run test once
  ```bash
  yarn test
  ```

- Watch the entire test suite and re-run when tests changed
  ```bash
  yarn test:watch
  ```
